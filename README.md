# Simple Todo App

The Simple Todo App is a task management application that allows users to create, organize, and track their daily tasks. It includes ways to add and delete tasks and due dates.

## Functionality

Users can create new to-do items through a popup modal. Input fields includes a required task input, an optional due date, and form validation. Upon form submission tasks are added to the to-do list and can be marked completed or uncompleted by checking or unchecking the checkboxes. Each task is assigned a unique ID generated with the UUID package, ensuring that every item can be individually managed. Users can also delete tasks they no longer need, and the form resets automatically after a successful submission.

## Technology

This project was built using HTML, CSS, and JavaScript, following the BEM methodology for organizing styles. Object-Oriented Programming (OOP) principles were used to create reusable Todo and FormValidator classes that manage task rendering and form validation functionality.
